There is no reason to concatenate literal strings. Doing so is an exercise in reducing code readability. Instead, the strings should be combined.

Noncompliant Code Example

$msg = "Hello " . "${name}" . "!";  // Noncompliant

Compliant Solution

$msg = "Hello ${name}!";
