In the interests of clean code, getters should be implicit, rather than explicit for properties that have only getters.

Noncompliant Code Example

var luckyNumber: Int {
  get {  // Noncompliant
    return 7
  }
}

Compliant Solution

var luckyNumber: Int {
  return 7
}
