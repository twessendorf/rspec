Memory that is allocated with new T[n] must be freed with delete[]. Leave out the [], and the likely result is heap corruption or, as a best-case scenario, premature program termination.

Noncompliant Code Example

char *cp = new char[10];
// ...
delete cp;  // Noncompliant

Compliant Solution

char *cp = new char[10];
// ...
delete[] cp;
