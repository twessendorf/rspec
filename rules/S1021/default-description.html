The explicit keyword prevents the constructor from being used to implicitly convert from a fundamental type to the class type.

Noncompliant Code Example

class C
{
public:
  C ( int32_t a ) // Noncompliant
  {
  }
};

Compliant Solution

class D
{
public:
  explicit D ( int32_t a )
  {
  }
};

  • MISRA C++:2008, 12-1-3
