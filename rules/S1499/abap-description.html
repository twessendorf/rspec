SELECT * should be avoided because it releases control of the returned columns and could therefore lead to errors and potentially to performance issues.

Noncompliant Code Example

SELECT *
       FROM persons
       INTO newyorkers
       WHERE city = 'NEW YORK'.

Compliant Solution

SELECT firstname, lastname
       FROM persons
       INTO newyorkers
       WHERE city = 'NEW YORK'.
