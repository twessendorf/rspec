Sharing some naming conventions is a key point to make it possible for a team to efficiently collaborate. This rule allows to check that all variable names match a provided regular expression.

Implementation Specification

(visible only on this page)

Message

Rename this variable "XXXX" to match the regular expression ${format}

Parameters

  • key = format Description: Regular expression used to check the names against.

Default Value : [A-Z0-9_-]++|[a-z0-9_-]++

C/C++ Default Value: [a-z][a-z_]*

Default value PL/I: [a-zA-Z0-9_]++

Swift Default Value: ^[a-z][a-zA-Z0-9]*$
