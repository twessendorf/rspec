Cognitive Complexity is a measure of how hard the control flow of a method is to understand. Methods with high Cognitive Complexity will be difficult to maintain.

  • Cognitive Complexity

Implementation Specification

(visible only on this page)

Message

Refactor this {0} to reduce its Cognitive Complexity from {1} to the {2} allowed.

Where {0} is the name of the syntax being analyzed, {1} the cognitive complexity and {2} the maximum authorized complexity.

Parameters

  • key = threshold

    • default: 15

    • description: The maximum authorized complexity.

  • key: propertyThreshold

    • default: 3

    • description: The maximum authorized complexity in a property.

Highlighting

  • primary: Method/function name

  • secondary: Keywords & operators that cause increments
