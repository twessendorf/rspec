This rule allows banning certain methods.

Noncompliant Code Example

Given parameters:

  • className:java.lang.String

  • methodName: replace

  • argumentTypes: java.lang.CharSequence, java.lang.CharSequence

String name;
name.replace("A","a");  // Noncompliant

Implementation Specification

(visible only on this page)

Message

Remove this forbidden call

Parameters

  • key = className

    • description: Name of the class whose method is forbidden

  • key: methodName

    • description: Name of the forbidden method

  • key: argumentTypes

    • description: Comma-delimited list of argument types, E.G. java.lang.String, int[], int

  • key: allOverloads

    • description: Boolean. Set to true to flag all overloads regardless of parameter type (default: false)
