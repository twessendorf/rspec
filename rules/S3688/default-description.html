This rule allows banning some modules.

Noncompliant Code Example

With moduleName configured with [a-zA-Z-]*UT123[a-zA-Z]*:

CALL UT123.
CALL UT123L.
CALL UT123-L.
CALL WS-UT123.

Implementation Specification

(visible only on this page)

Message

Remove the use of this forbidden module. Use {replacementModuleName} instead.

Parameters

  • key = moduleName

    • description: Name of the forbidden module. Use a regex to forbid more widely multiple modules sharing the same root name.

  • key: replacementModuleName

    • description: Name of the module to use as a replacement of the forbidden one.

Highlighting

First use of forbidden class
