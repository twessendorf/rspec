Every function should be commented to explain its goal and how it works. This non-empty comment must be located before the function definition.

Noncompliant Code Example

FUNCTION my_function.
  ...
ENDFUNCTION.

Compliant Solution

* here is
* my comment
FUNCTION my_function.
  ...
ENDFUNCTION.
