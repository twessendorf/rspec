Shared coding conventions allow teams to collaborate efficiently. This rule checks that all constant names match a provided regular expression.

Noncompliant Code Example

With the default regular expression ^[A-Z][A-Z0-9]*(_[A-Z0-9]+)*$:

define("const1", true);

class Foo {
    const const2 = "bar";
}

Compliant Solution

define("CONST1", true);

class Foo {
    const CONST2 = "bar";
}

Message

Rename this constant "XXX" to match the regular expression ${format}.

Parameters

  • key = format Description: Regular expression used to check the constant names against.

Default Value: ^[A-Z](_[A-Z0-9]+)$

Python default: [A-Z_][A-Z0-9_]*)|(.*$
